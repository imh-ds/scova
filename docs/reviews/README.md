# Stage 3 independent review protocol

Stable promotion requires two independent reviewers with expertise in
semiparametric or causal-inference theory. Reviewers should not be authors of
the derivation or implementation under review.

Each reviewer receives the theory appendix, notation map, implementation,
identity tests, simulation specification, and prior review resolutions. They
must address identification, tangent-space completeness, the one-step
remainder, finite-grid bootstrap validity, reductions, and public claims.

Copy `review-template.md` to `review-1.md` or `review-2.md`. A sign-off is valid
only when reviewer identity and date are recorded, every comment has a final
disposition, and the final decision is `approved`.
